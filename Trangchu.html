MODULE HTML

HTML (viết tắt của từ Hypertext Markup Language) hay là Ngôn ngữ đánh dấu siêu văn bản. Sử dụng HTML để xây dựng và cấu trúc lại các thành phần trong website hoặc ứng dụng.

MODULE CSS

CSS là chữ viết tắt của Cascading Style Sheets, nó là một ngôn ngữ được sử dụng để tìm và định dạng lại các phần tử được tạo ra bởi các ngôn ngữ đánh dấu (HTML).

MODULE JAVASCRIPT

JavaScript là gì? JavaScript là ngôn ngữ lập trình được nhà phát triển sử dụng để tạo trang web tương tác. Từ làm mới bảng tin trên trang mạng xã hội đến hiển thị hình ảnh động và bản đồ tương tác, các chức năng của JavaScript có thể cải thiện trải nghiệm người dùng của trang web.
